## 1. 设计原则

### 1.1 视觉设计原则

- **简洁明了**：减少视觉噪音，突出核心数据
- **信息层级**：重要信息优先展示，使用视觉层级引导用户
- **一致性**：统一的颜色、字体、组件风格
- **亲和力**：使用温暖的木色调，营造装修家居氛围

### 1.2 交互设计原则

- **快速录入**：最少点击完成常用操作（如记账）
- **实时反馈**：操作即时响应，数据实时更新
- **容错性**：支持撤销、编辑，防止误操作
- **渐进式披露**：基础功能前置，高级功能收起

### 1.3 色彩系统

```
主色调（Primary）    #D4A574  温暖木色
成功色（Success）    #4CAF50  绿色（进度完成）
警告色（Warning）    #FF9800  橙色（预算预警）
危险色（Error）      #F44336  红色（超支/延期）
中性色（Gray）       #9E9E9E  灰色（辅助信息）

背景色：
  - Primary Background    #FFFFFF  白色
  - Secondary Background  #F5F5F5  浅灰
  - Card Background       #FAFAFA  卡片底色
```

### 1.4 字体规范

```
标题：SF Pro Display / PingFang SC
  - H1: 28pt, Bold
  - H2: 22pt, Semibold
  - H3: 18pt, Semibold

正文：SF Pro Text / PingFang SC
  - Body: 16pt, Regular
  - Caption: 14pt, Regular
  - Small: 12pt, Regular

数字：SF Pro Rounded (更友好)
  - 金额/数据: 20pt+, Medium/Bold
```

## 4. 交互说明

### 4.1 全局交互

#### 4.1.1 下拉刷新
- 所有列表页面支持下拉刷新
- 刷新动画：转圈加载指示器
- 刷新完成后自动收起

#### 4.1.2 侧滑操作
```
支持侧滑的场景：
- 支出记录：左滑显示 [编辑] [删除]
- 任务列表：左滑显示 [编辑] [完成] [删除]
- 材料清单：左滑显示 [编辑] [改状态] [删除]
```

#### 4.1.3 长按操作
```
支持长按的场景：
- 图片：长按显示 [保存] [分享] [删除]
- 列表项：长按进入多选模式
- 阶段卡片：长按拖动调整顺序
```

#### 4.1.4 搜索
```
搜索交互：
- 点击搜索框自动聚焦弹出键盘
- 实时搜索（输入即搜索）
- 搜索历史记录（最近5条）
- 支持清空搜索内容
```

### 4.2 数据录入优化

#### 4.2.1 快速记账
```
优化路径：
1. 首页「记一笔」按钮 → 直接打开记账页
2. 预算页面悬浮按钮 → 快速记账
3. 3D Touch 应用图标 → 快捷记账

预填充优化：
- 记住上次选择的分类
- 自动填充当前日期
- 常用备注快速选择
```

#### 4.2.2 智能识别
```
OCR 识别流程：
1. 拍照/选择图片
2. 自动识别金额、日期、商家信息
3. 用户确认或修改
4. 保存记录

识别准确度提示：
- 高置信度：直接填充
- 低置信度：标黄提示用户确认
```

#### 4.2.3 批量操作
```
支持批量的场景：
- 导出多笔支出记录
- 删除多个任务
- 标记多个任务完成
- 分享多张装修照片

交互流程：
1. 长按进入多选模式
2. 勾选目标项
3. 底部显示操作按钮
4. 确认执行操作
```

### 4.3 数据可视化交互

#### 4.3.1 图表交互
```
支持的交互：
- 点击图表数据点查看详情
- 双指缩放调整时间范围
- 左右滑动切换时间段
- 图例点击显示/隐藏数据系列
```

#### 4.3.2 进度条交互
```
进度条状态：
- 绿色：0-70%（正常）
- 橙色：70-90%（注意）
- 红色：90-100%（预警）
- 深红：>100%（超标）

交互：
- 点击进度条查看详细数据
- 长按显示具体百分比
```

### 4.4 通知与提醒

#### 4.4.1 推送通知
```
通知场景：
- 预算达到预警阈值（80%、90%、100%）
- 任务即将到期（提前1天、3天）
- 付款节点提醒（提前3天）
- 材料到货通知
- 验收日期提醒

通知设置：
- 允许用户自定义提醒时间
- 支持静音时段设置
- 区分通知优先级
```

#### 4.4.2 应用内提示
```
Toast 提示：
- 操作成功/失败反馈
- 数据同步状态
- 网络连接状态

Alert 弹窗：
- 删除确认
- 重要操作二次确认
- 错误详情展示

底部 Sheet：
- 筛选选项
- 排序选项
- 更多操作菜单
```
