# Panda 装修管家 - 产品需求文档

> 让装修不再焦虑，每一分钱都花得明白

## 1. 产品概述

### 1.1 产品愿景

打造一款简洁易用的装修管理工具，帮助业主轻松掌控装修预算和进度，告别装修焦虑。

### 1.2 产品定位

面向首次装修或有装修需求的业主，提供预算管理、进度追踪、材料记录等一站式装修管理服务。

### 1.3 目标用户

| 用户类型 | 特征 | 核心需求 |
|---------|------|---------|
| 首次装修业主 | 缺乏经验，容易超预算 | 预算规划、避坑指南 |
| 有经验业主 | 追求效率，注重细节 | 高效记录、对比分析 |
| 出租房业主 | 控制成本，快速装修 | 简化流程、成本控制 |

## 2. 核心功能

### 2.1 预算管理 💰

#### 2.1.1 总预算设置
- 设置装修总预算
- 按房屋面积智能推荐预算范围（简装/中等/精装）
- 预算预警阈值设置（如达到80%时提醒）

#### 2.1.2 分类预算
预设分类（可自定义）：
```
├── 设计费
├── 硬装
│   ├── 拆改
│   ├── 水电
│   ├── 泥瓦
│   ├── 木工
│   └── 油漆
├── 主材
│   ├── 地板/瓷砖
│   ├── 门窗
│   ├── 橱柜
│   ├── 卫浴
│   └── 灯具
├── 软装
│   ├── 家具
│   ├── 家电
│   ├── 窗帘
│   └── 装饰品
└── 其他/应急
```

#### 2.1.3 支出记录
- 快速记账：金额 + 分类 + 备注 + 照片
- 支持拍照识别小票/发票金额
- 付款方式记录（全款/定金/尾款）
- 关联供应商/商家

#### 2.1.4 预算分析
- 实时预算使用率展示
- 分类支出占比饼图
- 超支预警提醒
- 与同类型装修预算对比参考

### 2.2 进度管理 📅

#### 2.2.1 装修阶段
预设标准装修流程：
```
1. 前期准备（量房、设计、预算）
2. 拆改阶段
3. 水电改造
4. 泥瓦工程
5. 木工工程
6. 油漆工程
7. 安装阶段（橱柜、门、地板等）
8. 软装入场
9. 保洁验收
10. 通风入住
```

#### 2.2.2 任务清单
- 每个阶段下的具体任务
- 任务状态：待开始 / 进行中 / 已完成 / 有问题
- 任务关联：负责人、预计时间、实际时间
- 支持自定义添加任务

#### 2.2.3 时间线视图
- 甘特图展示整体进度
- 里程碑标记
- 延期预警
- 历史进度记录

#### 2.2.4 验收清单
- 每阶段验收要点提示
- 验收拍照记录
- 问题标记与跟踪

### 2.3 材料管理 🧱

#### 2.3.1 材料清单
- 按房间/类型分类管理
- 记录：品牌、型号、规格、单价、数量、总价
- 购买状态：待购 / 已购 / 已到货 / 已安装

#### 2.3.2 价格对比
- 同类材料多商家比价
- 记录不同渠道报价
- 标记最终选择

#### 2.3.3 材料计算器
- 瓷砖用量计算（输入面积，考虑损耗）
- 油漆用量计算
- 地板用量计算
- 电线/水管估算

### 2.4 合同与文档 📄

#### 2.4.1 合同管理
- 拍照存储合同
- 合同关键信息提取（金额、工期、付款节点）
- 付款提醒

#### 2.4.2 重要文档
- 户型图
- 设计图纸
- 报价单
- 保修卡

### 2.5 通讯录 👥

- 装修公司/工长
- 设计师
- 各工种师傅
- 材料供应商
- 备注评价与推荐指数

### 2.6 装修日记 📸

- 按时间线记录装修过程
- 支持图片、视频、文字
- 前后对比展示
- 可选择性分享

## 5. 技术架构建议

### 5.1 技术选型

| 层级 | 技术方案 | 说明 |
|-----|---------|------|
| iOS 端 | SwiftUI + Swift | 现代化UI框架，声明式编程 |
| 架构模式 | MVVM + Clean Architecture | 易于测试和维护 |
| 本地存储 | SwiftData / Core Data | Apple 原生方案 |
| 云同步 | CloudKit | iCloud 同步，免服务器 |
| 图表库 | Swift Charts | iOS 16+ 原生支持 |
| 图片存储 | CloudKit Assets | 随 iCloud 同步 |

## 6. 设计规范

### 6.1 设计原则
- **简洁**：减少操作步骤，快速记录
- **清晰**：信息层级分明，一目了然
- **温暖**：缓解装修焦虑，传递安心感

### 6.2 色彩方案
- 主色：温暖的木质色系（体现家的感觉）
- 辅助色：清新绿色（进度/完成状态）
- 警示色：柔和橙色（预警/超支）
- 背景：米白/浅灰（舒适阅读）

### 6.3 图标风格
- 线性图标为主
- 圆润友好的风格
- 支持 SF Symbols

## 7. 竞品分析

| 产品 | 优势 | 不足 |
|-----|------|------|
| 装修记账本 | 简单易用 | 功能单一，无进度管理 |
| 好好住 | 社区活跃，灵感丰富 | 管理工具弱 |
| 土巴兔 | 功能全面 | 商业化重，广告多 |
| 住小帮 | 内容丰富 | 工具属性弱 |

**我们的差异化**：
- 专注管理工具，不做内容平台
- 本地优先，保护隐私
- 简洁设计，降低使用门槛
- 离线可用，数据自主

## 8. 商业模式

### 8.1 初期策略
- 核心功能免费
- 无广告
- 积累用户和口碑

### 8.2 后期变现（可选）
- **高级功能订阅**：数据分析报告、多项目管理、家庭共享
- **装修服务对接**：精选服务商推荐（佣金模式）
- **数据增值**：匿名数据分析报告（B端）

## 9. 成功指标

### 9.1 MVP 阶段
- 日活用户（DAU）
- 记账频率（每周 >= 3 次为活跃）
- 用户留存（7日、30日）
- App Store 评分 >= 4.5

### 9.2 增长阶段
- 月活用户（MAU）
- 用户推荐率（NPS）
- 功能使用率
- 付费转化率（如有）

## 10. 风险与挑战

| 风险 | 应对策略 |
|-----|---------|
| 用户装修周期短，用完即走 | 提供长期价值（家居维护、二次装修） |
| 记录门槛高，用户懒得记 | 简化操作，智能识别，语音输入 |
| 竞品功能同质化 | 深耕用户体验，打造口碑 |
| 数据同步问题 | CloudKit 成熟方案，本地优先 |
